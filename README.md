# VK Video Enhancer

Этот репозиторий содержит userscript для улучшения опыта просмотра видео на сайтах VK.com и VK Video.

---

## ⚠️ Важно

Используйте скрипт **на свой страх и риск**.  
Вы вмешиваетесь в работу стороннего веб-приложения (VK), и после обновлений сайта функциональность может измениться или перестать работать.

Исходный код полностью открыт и доступен в этом репозитории — вы можете проверить, что именно делает скрипт.

---

## 🚀 Возможности

Скрипт предназначен для устранения визуальных помех при просмотре видео:

* **Убирает размытие (blur)** с превью видеороликов  
* **Скрывает оверлеи** и затемняющие плашки (в том числе возрастные ограничения)  
* **Работает на `vk.com` и `vkvideo.ru`**  
* **Поддерживает динамический контент** (лента, рекомендации, подгрузка при скролле)  
* Использует **runtime-патчинг (webpack)** — работает на уровне React, а не просто DOM

## 🛠️ Как установить

### 1. Установите менеджер скриптов

Для работы userscript'ов вам понадобится расширение-менеджер.

Рекомендуется использовать **Tampermonkey**:

* Chrome: https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo  
* Firefox: https://addons.mozilla.org/firefox/addon/tampermonkey/  

(Также доступен для Edge, Safari и Opera)

---

### 2. ВАЖНО: включите доступ к сайтам

После установки Tampermonkey:

1. Откройте настройки расширения  
2. Найдите раздел доступа к сайтам  
3. Установите:
   - **"Разрешить на всех сайтах"**  
   или  
   - добавьте вручную:
     - `vk.com`
     - `vkvideo.ru`

👉 Без этого скрипт может просто не запускаться

---

### 3. Установите скрипт

1. Перейдите по ссылке установки  
2. Tampermonkey откроет страницу установки  
3. Нажмите **"Установить"**

Скрипт начнёт работать автоматически

---

## 🔄 Обновления

Скрипт поддерживает **автоматическое обновление**.

Tampermonkey:
- проверяет новую версию по `@updateURL`
- устанавливает обновления автоматически

👉 Важно: обновления применяются только при изменении версии (`@version`)

VK активно обновляет свою архитектуру (React + webpack), поэтому:

- скрипт может временно перестать работать  
- потребуется обновление логики  

Репозиторий поддерживается и будет обновляться по мере необходимости
